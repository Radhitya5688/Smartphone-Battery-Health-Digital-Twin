# Smartphone Battery Health Digital Twin

## Overview

This project creates a Digital Twin of smartphone battery behavior using smartphone specification data and machine learning.

The system estimates:

- Battery Health
- Charge Cycles
- Device Temperature
- Voltage
- Health Status
